Clone project:

    $ git clone https://github.com/diogoribeiro1/softwell-estagio.git
    
Run project:

    - Eclipse - 
    1- clique no projeto com o botao direito do mouse 
    2- va ate Run As -> Run Configurations -> Maven Build -> nome do projeto 
    3- digite em goal tomcat6:run
    4- apply -> run.
